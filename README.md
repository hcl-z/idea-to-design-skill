# Idea to UI Design Skill

[English Version](./README.en.md)

一个将用户模糊想法转化为前端 UI / 交互设计文档的 Skill。

# 工作流程

本 Skill 的主要流程基于 superpower 的 `brainstorming` skill 演化而来，保留了它的核心工作流：

- 先理解上下文
- 一次只问一个关键问题
- 提出 2-3 个方向并比较取舍
- 分段呈现设计并获得确认
- 产出设计文档
- 在需要时再交给后续规划或实现阶段

## 当前目标

这个 Skill 聚焦的是 `idea -> 前端 UI / 交互设计`，重点产出：

- 完整页面清单
- 每个页面的布局与模块结构
- 用户流程
- 关键交互与状态设计
- 视觉方向与精简的 design system 说明

## 不包含的内容

这个 Skill 不会输出以下内容：

- 后端架构
- API 设计
- 数据库或数据模型设计
- 具体代码逻辑
- 工程实现方案

## 输出形态

最终产物是一个面向前端界面设计的 Markdown 设计文档，而不是技术实现文档。
